# 순서도의 기호
- 프로그램 순서도(program flowchart)
    - 기호와 도형으로 프로그램의 흐름을 표시해둔 차트
    - 순서도 기호(flowchart symbol)

- 순서도 작성법
    - 순서도는 위에서 아래로, 왼쪽에서 오른쪽으로 작성
    - 각 기호는 한 가지 기능만 수행
    - 화살표는 흐름의 방향을 명확히 표시
    - 조건문은 결정 기호를 사용하여 '예'와 '아니오' 경로 구분

## 주요 순서도 기호

| 구성 | 내용 | 모양 |
| --- | --- | --- |
| 데이터(data) | 데이터 입출력 | <img src="assets/flowchart-symbols/data.svg" alt="데이터 기호" width="140"> |
| 처리(process) | 데이터 연산, 저장 등의 처리 | <img src="assets/flowchart-symbols/process.svg" alt="처리 기호" width="140"> |
| 미리 정의한 처리(predefined process) | 서브 루틴 및 모듈 등 다른 곳에서 이미 정의한 하나 이상의 연산 또는 명령어들로 이루어진 처리 | <img src="assets/flowchart-symbols/predefined-process.svg" alt="미리 정의한 처리 기호" width="140"> |
| 판단(decision) | 조건에 따라 데이터의 흐름을 결정 | <img src="assets/flowchart-symbols/decision.svg" alt="판단 기호" width="140"> |
| 단말(terminator) | 프로그램 흐름의 시작과 끝 | <img src="assets/flowchart-symbols/terminator.svg" alt="단말 기호" width="140"> |
| 선(line) | 기호 사이의 연결 관계를 표시 | <img src="assets/flowchart-symbols/line.svg" alt="선 기호" width="140"> |
