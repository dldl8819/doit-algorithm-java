# 세 값의 최댓값
- 변수 a, b, c에 들어가는 값은 키보드에서 입력한 값이며
- 이 3개의 값 중 최댓값을 변수 max에 담는다.
- 최댓값을 구하는 순차적(concatenation) 구조
    - 단순 대입
        - max에 a 값을 넣는다.
    - if문 : 선택(selection) 구조
        - b 값이 max보다 크면 max에 b 값을 넣는다.
        - c 값이 max보다 크면 max에 c 값을 넣는다.
## 키보드로 숫자와 문자열 입력하기
```java
import java.util.Scanner;
```
- 전체 프로그램 앞머리(클래스 선언보다 앞)에 작성
- java.util 패키지에 있는 Scanner 클래스를 프로그램에 포함시킴
```java
Scanner stdIn = new Scanner(System.in);
```
- main 메서드 맨 앞에 작성
- `System.in`은 키보드와 연결된 표준 입력 스트림(standard input stream)
```java
stdIn.nextInt()
```
- 키보드로 입력한 값 중 `정수값`을 얻어냄
    - int형의 범위 -2,147,483,648 ~ 2,147,483,647로 제한
- 알파벳이나 기호 등은 입력 불가
- `stdIn`은 키보드와 연결된 표준 입력 스트림(`System.in`)에서 문자나 숫자를 꺼내는 장치 역할로 변수명 stdIn은 다른 이름으로 변경 가능

## 자료형에 따라 다른 입력 메서드 사용
- `Scanner`는 입력값의 자료형에 따라 호출 메서드가 다르다.
- 입력받으려는 자료형과 메서드가 맞지 않으면 `InputMismatchException`이 발생할 수 있다.

| 메서드 | 자료형 | 입력값 범위/형태 |
|---|---|---|
| `nextBoolean()` | `boolean` | `true` 또는 `false` |
| `nextByte()` | `byte` | `-128` ~ `127` |
| `nextShort()` | `short` | `-32768` ~ `32767` |
| `nextInt()` | `int` | `-2147483648` ~ `2147483647` |
| `nextLong()` | `long` | `-9223372036854775808` ~ `9223372036854775807` |
| `nextFloat()` | `float` | 약 `+-3.40282347E+38` ~ `+-1.40239846E-45` |
| `nextDouble()` | `double` | 약 `+-1.79769313486231570E+308` ~ `+-4.94065645841246544E-324` |
| `next()` | `String` | 문자열(스페이스, 줄 바꿈 문자로 구분) |
| `nextLine()` | `String` | 줄바꿈 전까지 한 줄 전체 |

## 매개변수
- 메서드를 정의할 때 메서드에 전달되는 값을 저장하기 위해 변수(variable)를 선언한다.
- 이를 매개변수(parameter) 또는 형식매개변수(formal parameter)라 한다.
- 형식매개변수를 가인수(임시 인수)라 하고 메서드를 호출할 때 사용하는 매개변수의 값을 실인수(actual argument)라 한다.
- 메서드를 정의할 땐 `매개변수`, 메서드를 호출할 때는 `실인수`
  - `매개변수`: 메서드 선언부 괄호 안에 있는 변수 이름
  - `실인수`: 메서드 호출 시 실제로 넘기는 값(리터럴, 변수, 식)
- 호출될 때 실인수의 값이 매개변수에 대입된다.

```java
static int max3(int a, int b, int c) {   // a, b, c = 매개변수
    int max = a;
    if (b > max) max = b;
    if (c > max) max = c;
    return max;
}

int x = 3, y = 8, z = 5;
int result = max3(x, y, z);              // x, y, z = 실인수
int result2 = max3(10, 20, 15);          // 10, 20, 15 = 실인수
```
