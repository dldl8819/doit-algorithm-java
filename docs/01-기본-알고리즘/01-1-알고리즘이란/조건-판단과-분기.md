# 조건 판단과 분기
- 연산자(operator)
    - +, - 등의 연산 기호
    - a > b 에서 `>`    
- 피연산자(operand)
    - 연산의 대상
    - `a`, `b`
## 연산자의 종류
- 피연산자의 수에 따라 3종류로 분류됨
    - 단항 연산자(unary operator)
        - 피연산자 1개
        - a++
    - 2항 연산자(binary operator)
        - 피연산자 2개
        - a < b
    - 3항 연산자(ternary operator)
        - 피연산자 3개
        - a ? b : c
            - a가 true이면 b를 반환하고 false면 c를 반환
            - `?`, `:`는 자바에서 유일한 3항 연산자
            ```java
            a = ( x > y ) ? x : y;
            ``` 
            - `x > y`가 `true`면 x를 반환
